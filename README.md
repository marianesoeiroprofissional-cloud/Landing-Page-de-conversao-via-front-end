Olá! Bem-vindo(a) ao meu segundo projeto: uma landing page de alta conversão para um infoproduto, desenvolvida
via JavaScript.

O principal intuito deste trabalho é externalizar meus conhecimentos em Design e lógica front-end. Vale ressaltar
que o curso abordado no site é fictício, criado com o objetivo de guiar o usuário através de uma interface até um
formulário de acesso, que o direciona diretamente ao WhatsApp para uma possível venda.

A página conta com elementos visualmente atraentes, um design clean, interface intuitiva e totalmente responsiva
juntamente com botões de chamada para ação (CTAs). Para a validação dos scripts e comportamento das rotas em 
tempo de execução, o projeto foi estruturado e testado localmente em ambiente de desenvolvimento web através do XAMPP.
