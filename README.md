🚀 Meu 2º Projeto: Landing Page de Alta Conversão
Olá! Bem-vindo(a) ao meu segundo projeto: uma landing page de alta conversão para um infoproduto, desenvolvida via JavaScript.

🎯 O Objetivo do Projeto
Propósito: O principal intuito deste trabalho é externalizar meus conhecimentos em Design e lógica front-end.

Fluxo de Conversão: Vale ressaltar que o curso abordado no site é fictício, criado com o objetivo de guiar o usuário através de uma interface até um formulário de acesso, que o direciona diretamente ao WhatsApp para uma possível venda.

💻 Diferenciais Técnicos & Interface
A página conta com elementos visualmente atraentes, um design clean, interface intuitiva e totalmente responsiva juntamente com botões de chamada para ação (CTAs).

Ambiente de Testes: Para a validação dos scripts e comportamento das rotas em tempo de execução, o projeto foi estruturado e testado localmente em ambiente de desenvolvimento web através do XAMPP.
